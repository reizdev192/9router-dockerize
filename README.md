## 🇬🇧 English Version

### 📖 Introduction
A production-ready Dockerized setup for **9router**. This repository provides everything you need to run `9router` in a containerized environment with persistent storage, automated environment variable injection, and seamless upgrade commands.

### ✨ Features
- **Containerized Environment:** Runs cleanly inside an isolated, lightweight Node.js Alpine container.
- **Persistent Storage:** Volumes are pre-configured so that your databases (`db.json`) and usage logs are safely stored on the host machine. You will never lose data if the container is destroyed.
- **Easy Configuration:** Out-of-the-box `.env.docker` configuration template included.
- **Zero-downtime Upgrades:** Simple commands to fetch and rebuild the latest `9router` version from npm.

### 🚀 Quick Start

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd 9router
   ```

2. **Configure Environment:**
   Review and update the variables in `.env.docker`. It is highly recommended to change `JWT_SECRET` and `API_KEY_SECRET` before deploying to production.

3. **Start the Container:**
   ```bash
   docker compose up -d
   ```
   *The service will be available at `http://localhost:20128` (or your configured port).*

### 📂 Volumes
- `./9router-data`: Persists the primary application state (`db.json`) and run history.
- `./9router-logs`: (If ENABLE_REQUEST_LOGS is true) Persists request logic logs generated by the app.

### 🔄 Upgrading to the Latest Version
To upgrade `9router` to the newest package available on npm, simply trigger a fresh build without cache to force a re-download:
```bash
# 1. Force docker to rebuild without caching the old npm version
docker compose build --no-cache

# 2. Restart your container with the new image
docker compose up -d
```

## 🇻🇳 Phiên bản Tiếng Việt

### 📖 Giới thiệu
Bộ cấu hình triển khai siêu cấp chuẩn chỉ để khởi chạy **9router** bằng thẻ Docker. Kho lưu trữ này cung cấp sẵn đầy đủ "đồ chơi" giúp bạn chạy `9router` ổn định 24/7, lưu trữ dữ liệu an toàn tuyệt đối và cực kỳ dễ bảo trì.

### ✨ Tính năng nổi bật
- **Môi trường cách ly:** Hoạt động trơn tru trong nền tảng Node.js Alpine siêu nhẹ bé tí xíu, không rác máy chủ.
- **Dữ liệu bất tử:** Volume tự động mount thư mục cấu hình ra ngoài máy thật. Dữ liệu (`db.json`) và lịch sử dùng sẽ không bao giờ mất kể cả khi bạn lỡ tay xóa container.
- **Cấu hình nhàn tênh:** File `.env.docker` chuẩn khung với đầy đủ template biến môi trường cần thiết từ tài liệu gốc.
- **Nâng cấp một chạm:** Vài dòng lệnh là lấy được phiên bản 9router mới nhất trên npm về máy.

### 🚀 Bắt đầu nhanh

1. **Tải mã nguồn về máy (Clone git):**
   ```bash
   git clone <your-repo-url>
   cd 9router
   ```

2. **Tùy chỉnh Cấu hình (Rất quan trọng):**
   Mở file `.env.docker` và cấu hình lại theo ý muốn. Hãy nhớ đổi các chuỗi bảo mật thật khó đoán, nhất là `JWT_SECRET` và `API_KEY_SECRET`.

3. **Khởi động Container:**
   ```bash
   docker compose up -d
   ```
   *Bùm! 💥 9router sẽ lập tức hoạt động tại cổng `http://localhost:20128`.*

### 📂 Thư mục lưu trữ (Volumes)
- `./9router-data`: Lưu trữ cấu hình chính thức yếu (`db.json`) và lịch sử hoạt động.
- `./9router-logs`: Nơi nhét file nhật ký (nếu bạn bật lưu log bên trong config).

### 🔄 Cách cập nhật lên bản mới nhất
Khi có bản cập nhật nâng cấp `9router` mới trên npm, bạn chỉ việc gõ Combo 2 lệnh thần thánh sau để ép Docker quăng đồ cũ và sắm diện mạo mới:
```bash
# 1. Ép Docker bỏ qua bộ nhớ đệm, lên chợ tải bản cài mới tinh
docker compose build --no-cache

# 2. Xây lại container thay thế phiên bản cũ
docker compose up -d
```
